# avalanchesass_utility_width
Width classes.

## Install
```
npm install avalanchesass_utility_width --save
```

## Basic usage
```css
/* Import the main file. */
@import 'avalanchesass_utility_width';

/* Import just the mixin file. */
@import 'avalanchesass_utility_width/mixin';
```
